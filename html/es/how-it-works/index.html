Como funciona el workflow renovado de WallCraft

La interfaz 2026.3.14 esta construida alrededor de un ciclo mas simple: importar, colocar, escalar, reordenar, previsualizar y aplicar.

Cinco pasos desde las imagenes al escritorio

El workflow actual es mas facil de entender y controlar.

01

Importar imagenes

Cargar una o varias fotos en la biblioteca del proyecto.

02

Construir la composicion

Colocar las imagenes dentro del escritorio virtual y elegir un layout.

03

Escalar sin distorsion

Usar el modo dedicado para cambiar el tamano respetando la proporcion.

04

Ajustar el orden visual

Enviar una imagen atras o delante cuando hay superposiciones.

05

Previsualizar y aplicar

Validar la vista multi-monitor y luego exportar o aplicar en Windows.
